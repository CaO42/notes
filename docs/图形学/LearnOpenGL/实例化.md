### 实例化

实例化是OpenGL为了实现Draw Call的合并而采用的技术

**适用场景：** 绘制大量完全相同、只是位置/旋转/缩放/颜色不同的网格（如：一片森林中的树木、草地、飞舞的粒子）。

```c++
// 顶点着色器数据
#version 330 core
layout (location = 0) in vec2 aPos;
layout (location = 1) in vec3 aColor;

out vec3 fColor;

uniform vec2 offsets[100];

void main()
{
    vec2 offset = offsets[gl_InstanceID]; // gl_InstanceID是内置变量，从0开始，当调用实例化的draw函数渲染每个图元时 获取其对应的索引
    gl_Position = vec4(aPos + offset, 0.0, 1.0);
    fColor = aColor;
}

// cpp代码
glBindVertexArray(quadVAO);// 该VAO只包含一个图元的数据
glDrawArraysInstanced(GL_TRIANGLES, 0, 6, 100);
```

### 实例化数组

实例化数组(Instanced Array)，它被定义为一个**顶点属性**（能够让我们储存更多的数据），仅在顶点着色器渲染一个新的实例时才会更新。
