## 操作系统原理 作业-6

**第八章 Main Memory（主存）与第九章 Virtual Memory（虚拟内存）** 参考教材：Operating System Concepts, 9th Edition (Silberschatz et al.)

> **说明：**以下题目依据主存管理、分页与虚拟内存核心知识点原创整理，适合作为课后作业。

一、选择题

1. 在支持运行时地址重定位的系统中，逻辑地址到物理地址的转换通常由（ ）完成。

   - A. 编译器
   - B. 链接器
   - C. 装入程序
   - D. 内存管理单元（MMU）

2. 使用基址寄存器（base register）和界限寄存器（limit register）进行内存保护时，合法逻辑地址 x 应满足（ ）。

   - A. 0 <= x < limit
   - B. base <= x < limit
   - C. x < base
   - D. x >= base + limit

3. 分页存储管理相对于可变分区连续分配的一个重要优点是（ ）。

   - A. 完全消除内部碎片
   - B. 不需要页表
   - C. 避免外部碎片，进程的物理页框不必连续
   - D. 每次地址访问都不需要硬件支持

4. 快表（TLB, Translation Look-aside Buffer）的主要作用是（ ）。

   - A. 保存所有磁盘块的索引
   - B. 缓存最近使用的页表项，加快地址转换
   - C. 替代主存保存用户数据
   - D. 执行页面置换算法

5. 关于分段（Segmentation）存储管理，下列说法正确的是（ ）。

   - A. 段是程序员可见的逻辑单位，如代码段、数据段和栈段
   - B. 所有段长度必须相同
   - C. 分段不会产生任何碎片
   - D. 分段地址中不需要段号

6. 请求分页（Demand Paging）系统中，缺页异常（page fault）通常发生在（ ）。

   - A. CPU 访问的页面当前不在物理内存中，页表项标记为无效
   - B. CPU 执行普通加法指令时
   - C. TLB 命中时
   - D. 进程只读已在内存中的页面时

7. 写时复制（Copy-on-Write）的基本思想是（ ）。

   - A. fork 后立即复制父进程的全部物理页面
   - B. 父子进程先共享只读页面，只有某一方写入时才复制该页面
   - C. 禁止用户进程写内存
   - D. 把所有页面预先写回磁盘

8. 最优页面置换算法（Optimal Page Replacement）的理论依据是（ ）。

   - A. 总是置换最早进入内存的页面
   - B. 总是置换最近最久未使用的页面
   - C. 置换未来最长时间不会再被访问的页面
   - D. 随机选择一个页面置换

9. 下列页面置换算法中，可能出现 Belady 异常的是（ ）。

   - A. FIFO
   - B. LRU
   - C. Optimal
   - D. 基于栈性质的算法

10. 系统发生抖动（Thrashing）的典型表现是（ ）。

   - A. CPU 利用率很高，缺页率很低
   - B. 进程频繁缺页，系统大量时间花在换页而非执行用户代码
   - C. 所有进程都只使用寄存器
   - D. TLB 命中率恒为 100%

二、简答题

1. 请比较首次适应（First Fit）、最佳适应（Best Fit）和最差适应（Worst Fit）三种连续内存分配算法，并说明外部碎片与内存紧缩（Compaction）的关系。
2. 请解释分页系统中的页（page）、页框（frame）、页表（page table）和页表项（PTE）。一次普通内存访问在没有 TLB 和有 TLB 的情况下分别需要哪些步骤？
3. 什么是请求分页？请描述从 CPU 访问一个不在内存中的页面开始，到该指令重新执行为止，操作系统处理缺页异常的大致流程。

三、分析题

1. 某分页系统页面大小为 1KB，页表如下。页号为逻辑地址除以页面大小的商，页内偏移为余数。

| **页号** | **有效位** | **页框号** |
| -------- | ---------- | ---------- |
| 0        | 1          | 5          |
| 1        | 1          | 9          |
| 2        | 0          | -          |
| 3        | 1          | 2          |
| 4        | 1          | 11         |
| 5        | 0          | -          |

**请回答：**

- （a）分别将逻辑地址 100、1350、3220、5200 转换为物理地址；若发生缺页异常，请指出。
- （b）若访问内存一次需要 100ns，TLB 查询需要 10ns，TLB 命中率为 80%，忽略缺页异常开销，计算有效访问时间（Effective Access Time）。
- （c）如果页表项中增加保护位和修改位，它们分别可以用于什么目的？

2. 某进程分配 3 个页框，页面访问序列为：7, 0, 1, 2, 0, 3, 0, 4, 2, 3, 0, 3, 2 请分别使用 FIFO、LRU 和 Optimal 页面置换算法模拟执行过程，给出每种算法的缺页次数，并简要比较三者差异。
