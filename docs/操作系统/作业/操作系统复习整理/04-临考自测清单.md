# 临考自测清单

用这份清单自测时，如果某一项不能在 1-2 分钟内讲清楚或算完，就回到对应章节复习。

## A. 基础概念

- [ ] 我能说出操作系统的三个主要目的。
- [ ] 我能解释内核模式和用户模式如何保护系统。
- [ ] 我能判断一条指令是否应为特权指令。
- [ ] 我能说明中断、陷阱、系统调用之间的关系。
- [ ] 我能说明 DMA 相比中断驱动 I/O 的优势。

## B. 操作系统结构

- [ ] 我能列举 OS 面向用户便利性的服务和面向系统效率的服务。
- [ ] 我能描述 API、运行时库、系统调用接口和内核之间的关系。
- [ ] 我能比较单体内核、微内核、分层结构、可加载模块。
- [ ] 我能从安全性、性能、可维护性分析一个服务应放内核态还是用户态。
- [ ] 我能解释 eBPF、XDP、map、verifier 的作用。

## C. 进程

- [ ] 我能区分程序和进程。
- [ ] 我能列出 PCB 的典型内容。
- [ ] 我能画出基本进程状态转换，并知道等待态不能直接到运行态。
- [ ] 我能分析 `fork()` 的返回值、变量副本和输出顺序。
- [ ] 我能区分僵尸进程和孤儿进程。
- [ ] 我能比较共享内存和消息传递。
- [ ] 我能解释循环缓冲区为什么通常只能用 `BUFFER_SIZE - 1` 个槽。

## D. CPU 调度

- [ ] 我能解释抢占式和非抢占式调度的区别。
- [ ] 我能画 FCFS、SJF、SRTF、RR、优先级调度的甘特图。
- [ ] 我能计算完成时间、周转时间、等待时间和平均值。
- [ ] 我能说明 RR 时间片过大/过小的问题。
- [ ] 我能模拟多级反馈队列中的队列降级和抢占。
- [ ] 我能解释优先级反转、硬实时/软实时、RMS、处理器亲和性。

## E. 同步与死锁

- [ ] 我能定义竞争条件和临界区。
- [ ] 我能解释互斥、前进、有界等待。
- [ ] 我能说明 Peterson 算法中 `flag[]` 和 `turn` 的作用。
- [ ] 我能解释为什么现代多处理器上还要考虑内存屏障。
- [ ] 我能比较关中断、自旋锁、硬件原子指令、信号量。
- [ ] 我能区分二值信号量和计数信号量。
- [ ] 我能说明读写器锁为什么仍可能死锁。
- [ ] 我能列出死锁四个必要条件。
- [ ] 我能用银行家算法计算 Need、安全序列和请求是否可分配。
- [ ] 我能写出单线桥问题的信号量伪代码思路。

## F. 内存与虚拟内存

- [ ] 我能比较 First Fit、Best Fit、Worst Fit。
- [ ] 我能解释外部碎片和 compaction。
- [ ] 我能区分页、页框、页表、页表项、TLB。
- [ ] 我能完成分页地址转换并判断缺页。
- [ ] 我能计算 TLB 有效访问时间。
- [ ] 我能描述缺页异常处理完整流程。
- [ ] 我能解释写时复制。
- [ ] 我能模拟 FIFO、LRU、Optimal 页面置换。
- [ ] 我能解释 Belady 异常和抖动。

## G. 文件、磁盘与 I/O

- [ ] 我能说明磁盘访问时间的三部分。
- [ ] 我能模拟 FCFS、SSTF、SCAN、C-SCAN 磁盘调度。
- [ ] 我能解释 SSTF 的吞吐量优势和饥饿风险。
- [ ] 我能说明 RAID 1 的镜像思想。
- [ ] 我能解释设备驱动、中断驱动 I/O、DMA 的关系。
- [ ] 我能说明 inode/FCB 中保存哪些元数据。
- [ ] 我能描述目录项、路径名和 inode 的关系。
- [ ] 我能比较连续分配、链式分配、索引分配。
- [ ] 我能计算一级/二级间接索引能表示的文件大小。
- [ ] 我能从位图中找空闲块、连续空闲块和索引分配方案。
- [ ] 我能说明日志文件系统和 fsck 如何降低断电不一致风险。

## 最后两小时复习顺序

1. 先做一遍 CPU 调度题：甘特图 + 平均等待/周转。
2. 再做银行家算法：Need、安全序列、Request 试探分配。
3. 接着做内存题：地址转换、EAT、页面置换。
4. 最后做文件/磁盘题：磁盘调度、inode 容量、位图分配。
5. 剩余时间快速扫 [03-概念速查表.md](03-概念速查表.md)，专门看易混概念。
