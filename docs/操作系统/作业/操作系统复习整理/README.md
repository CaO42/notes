# 操作系统课程作业复习整理

本文件夹根据当前目录下 1-7 七个作业子文件夹中的 Markdown 文件整理而成；PDF 和 DOCX 未读取，因为题目说明它们与 Markdown 内容相同。

建议阅读顺序：

1. [00-复习总览.md](00-复习总览.md)：快速把握七次作业覆盖范围、考察方式和复习优先级。
2. [01-按作业章节考点.md](01-按作业章节考点.md)：按作业编号逐章复习，适合对照课本章节查漏补缺。
3. [02-题型与计算模板.md](02-题型与计算模板.md)：集中整理手算题、程序分析题、算法模拟题的解题步骤。
4. [03-概念速查表.md](03-概念速查表.md)：临考前快速背诵和区分易混概念。
5. [04-临考自测清单.md](04-临考自测清单.md)：用“我是否会...”的方式做最后检查。

复习建议：

- 先用总览确认范围，再把“必须会手算”的模板题过一遍。
- CPU 调度、银行家算法、地址转换、页面置换、磁盘调度、inode/位图计算是最值得优先练的题型。
- 概念题不要只背定义，重点准备“比较类”和“为什么类”表达，例如单体内核 vs 微内核、共享内存 vs 消息传递、抢占式 vs 非抢占式、连续/链式/索引分配。
