# 我的笔记

欢迎来到我的个人笔记库。这里记录我在学习与工作中的点滴积累。

## 内容分类

- **人工智能** —— 复习资料，覆盖绪论、知识表示与推理、搜索技术、不确定性推理、决策树、机器学习、人工神经网络、深度学习和强化学习

  - **Unity** —— Unity Shader 与 ShaderLab
  - **专题探究** —— 小孔成像与景深、NeRF、后处理等前沿话题
- **近世代数** —— 群论基础，包含等价关系、群的概念、子群、同构、循环群、置换群等内容，附通俗学习教程
