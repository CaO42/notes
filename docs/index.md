# 我的笔记

欢迎来到我的个人笔记库。这里记录我在学习与工作中的点滴积累。

## 关于本站

- **内容**：数学、编程、阅读笔记等
- **特性**：完整支持 Markdown 与 LaTeX 数学公式
- **部署**：基于 MkDocs Material + GitHub Pages，push 后自动更新

## 快速开始

新增笔记只需在 `docs/` 目录下创建 `.md` 文件，例如：

```
docs/
├── 数学/
│   └── 微积分基础.md
└── 编程/
    └── python笔记.md
```

文件夹与文件会**自动出现在左侧导航**，无需手动维护目录。

> 推荐使用顶部搜索框快速定位内容，移动端点击左上角菜单展开导航。
