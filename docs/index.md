# 我的笔记

欢迎来到我的个人笔记库。这里记录我在学习与工作中的点滴积累。

## 内容分类

- **人工智能** —— 课程复习笔记，按主题分为搜索技术、概率推理、机器学习、深度学习四个板块，涵盖 A*、贝叶斯网络、决策树、CNN/RNN/LSTM、Transformer 等核心内容
- **图形学**
  - **FCG & GAMES101** —— 基于采样的光栅化、Blinn-Phong 光照、PBR 物理渲染等
  - **LearnOpenGL** —— 从基础管线到高级渲染（帧缓冲、阴影、延迟着色）和着色器进阶（几何着色器、法线贴图、实例化）
  - **Unity** —— Unity Shader 与 ShaderLab
  - **专题探究** —— 小孔成像与景深、NeRF、后处理等前沿话题
- **操作系统**
  - **Lab5 内核线程** —— 可变参数与 printf 实现、内核线程机制、线程调度与上下文切换、优先级调度算法
  - **Lab6 并发与锁机制** —— 自旋锁与信号量、CASLock、生产者-消费者同步、读者-写者问题
