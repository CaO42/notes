# 00 学习路线：先看清第一章在做什么

## 1. 本章为什么从等价关系开始

群论表面上研究“运算”，但它经常需要把对象分成一类一类。例如：

- 整数按模 $m$ 的余数分成 $m$ 类。
- 矩阵按相似关系分成若干类。
- 置换按共轭关系分成若干类。

所以教材先讲等价关系和集合分类。你可以把它理解成：以后我们不只关心单个元素，还会关心“哪些元素本质上算同一种”。

## 2. 第一章主线

第一章可以看成一条越来越具体的路线：

1. **1.1 等价关系与集合分类**  
   学会把集合按规则分堆。

2. **1.2 群的概念**  
   学会判断一个集合和一个运算是不是群。

3. **1.3 子群**  
   学会在大群中找小群。

4. **1.4 群的同构**  
   学会判断两个群是否“结构相同”。

5. **1.5 循环群**  
   学会研究由一个元素生成的群。这一类群最容易完全弄清楚。

6. **1.6 置换群与对称群**  
   学会用“排列重组”理解群，这是群论最重要的具体例子之一。

7. **1.7 置换在对称变换群中的应用**  
   选学。把置换群用于正多边形、正多面体、多项式的对称。

## 3. 常用记号

| 记号 | 含义 |
| --- | --- |
| $e$ | 乘法群的单位元 |
| $0$ | 加法群的单位元，也叫零元 |
| $a^{-1}$ | $a$ 的逆元 |
| $-a$ | 加法群中 $a$ 的负元 |
| $|G|$ | 群 $G$ 的阶，也就是元素个数 |
| $\operatorname{ord} a$ | 元素 $a$ 的阶 |
| $H<G$ | $H$ 是 $G$ 的子群 |
| $\langle S\rangle$ | 由集合 $S$ 生成的子群 |
| $\langle a\rangle$ | 由单个元素 $a$ 生成的循环子群 |
| $\mathbb Z_m$ | 模 $m$ 剩余类加群 |
| $U(m)$ | 模 $m$ 单位群，即模 $m$ 下与 $m$ 互素的剩余类 |
| $S_n$ | $n$ 次对称群 |
| $A_n$ | $n$ 次交代群，即 $S_n$ 中全体偶置换 |
| $D_n$ | 正 $n$ 边形的二面体群 |

## 4. 学群论时最重要的三个问题

看到一个新对象时，先问：

1. **集合是什么？**  
   例如 $\mathbb Z$、$\mathbb Z_m$、$GL_n(\mathbb R)$、$S_n$。

2. **运算是什么？**  
   是加法、乘法、矩阵乘法、映射合成，还是剩余类运算？

3. **运算是否能保持在集合内部？**  
   这叫封闭性。没有封闭性，连代数运算都还不是，更谈不上群。

## 5. 证明题的基本套路

本章最常见的证明题有四种。

### 证明是等价关系

依次证明：

1. 反身性：$a\sim a$。
2. 对称性：若 $a\sim b$，则 $b\sim a$。
3. 传递性：若 $a\sim b$ 且 $b\sim c$，则 $a\sim c$。

### 证明是群

通常检查：

1. 封闭性。
2. 结合律。
3. 单位元。
4. 逆元。

如果是某个已知群的子集，优先用子群判定法，不要从头检查群公理。

### 证明是子群

最常用的一步判定法：

非空子集 $H\subseteq G$ 是子群，只需证明对任意 $a,b\in H$，都有

$$
ab^{-1}\in H.
$$

加法群中对应为：

$$
a-b\in H.
$$

### 证明两个群同构

四步：

1. 定义映射 $\phi:G\to G'$。
2. 证明单射。
3. 证明满射。
4. 证明保持运算：$\phi(ab)=\phi(a)\phi(b)$。

## 6. 第一章的学习目标

学完这一章，你应该能做到：

- 判断常见集合和运算是否构成群。
- 判断一个子集是否是子群。
- 写出由元素生成的子群。
- 判断有限循环群的生成元和子群。
- 计算置换乘积、逆置换、轮换分解、置换阶和奇偶性。
- 理解为什么对称现象可以用群来描述。
