# 第一章 群：通俗学习教程

这是一份根据 `课程目录.md` 和 `Chapter_1_群.md` 整理的第一章学习教程。它不改动原教材文件，只作为独立学习材料放在本文件夹中。

## 怎么使用

建议按顺序学习。第一遍不要急着记住所有定理，先把每个概念回答成一句人话：

- 等价关系：把元素按某种“同类标准”分堆。
- 群：一个集合加上一种运算，运算像“可逆的加法或乘法”一样好用。
- 子群：大群里面仍然能独立运算的小群。
- 同构：两个群只是“名字和外观不同”，结构其实一样。
- 循环群：全群由一个元素反复运算生成。
- 置换群：研究“重新排列”的群，是群论最重要的具体模型之一。

## 学习节奏

如果你是第一次学近世代数，可以这样安排：

1. 第一天：读 `00`、`01`、`02`，重点理解“集合 + 运算”的语言。
2. 第二天：读 `03`、`04`，重点练习子群判定和同构证明四步法。
3. 第三天：读 `05`，把循环群当成第一章最核心的可完全掌握模型。
4. 第四天：读 `06`，多动手算置换乘法。
5. 第五天：读 `07` 和 `08`，把抽象概念和几何对称联系起来。
