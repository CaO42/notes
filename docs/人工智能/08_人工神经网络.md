### 神经元

- 1904年生物学家已经知晓**神经元**的组成结构
- 一个神经元通常具有多个**树突**，主要用来接受传入信息；而**轴突**只有一条，轴突尾端有许多**轴突末梢**，跟其他神经元的树突产生连接（**突触**），从而给其他多个神经元传递信号

### 人脑神经元的信息传递

- 当一个神经元接收的信号**累积到一定程度**时，它会释放神经递质，该神经递质会经过**树突**前端前往**轴突末梢**，依靠突触刺激下一个神经元，实现多个神经元间**信息传递**

### 人脑神经元模型MP

- 1943年，心理学家McCulloch和数学家Pitts参考了生物神经元的结构，发表了抽象的**神经元模型MP**
- 类比生物结构：输入可以类比为神经元的树突，而输出可以类比为神经元的轴突，计算则可以类比为细胞核。**神经元模型是一个包含输入，输出与计算功能的模型**

类比生物结构：输入可以类比为神经元的树突，而输出可以类比为神经元的轴突，计算则可以类比为细胞核。神经元模型是一个包含输入、输出与计算功能的模型。

注意中间的箭头线。这些线称为"连接"。每个上有一个"权值"。

[截图待插入: 神经元模型示意图，左侧为输入1/2/3经权值1/2/3连接到求和节点，再经非线性函数到输出；右侧为生物神经元结构图，标注树突、轴突末梢、细胞核、轴突]

### 人工神经元模型MP（计算过程）

一个典型的神经元模型，包含有**N个输入**，**1个输出**，以及**2个计算功能**，每个连接上有一个权值。

| 输入 | 权值 | 计算过程 | 输出 |
|------|------|----------|------|
| $a_1$ | $w_1$ | 求和 sum | $z$ |
| $a_2$ | $w_2$ | 非线性函数 $g$ | |
| $a_3$ | $w_3$ | | |

$$z = g(a_1 w_1 + a_2 w_2 + a_3 w_3)$$

### 人工神经元模型MP（计算公式）

神经元的计算公式：用 $\boldsymbol{a} = [a_1, a_2, a_3]^T$ 表示输入，用 $\boldsymbol{W} = [w_1, w_2, w_3]$ 表示权值，经过加权计算末端信号为 $\boldsymbol{W} \times \boldsymbol{a}$，再叠加非线性函数 $g$，即是输出值 $z$。

在MP模型里，**非线性函数 $g$** 设置是**符号函数（sgn）**。

由于起到类似神经元的"激活"作用，也称作**激活函数**（Active Function）。

$$\text{sgn}(x) = \begin{cases} 1, & x \geq 0 \\ -1, & x < 0 \end{cases}$$

$$z = g(a_1 w_1 + a_2 w_2 + a_3 w_3) = g(\boldsymbol{W} \times \boldsymbol{a})$$

[截图待插入: 左侧为神经元计算流程图（输入a1/a2/a3经权值w1/w2/w3到求和sum再到非线性函数g到输出z）；右侧为sgn函数图像，阶梯状，x≥0时y=1，x<0时y=-1]

对神经元模型的图进行简化：将求和函数sum与激活函数sgn合并到一个圆圈里，代表神经元的内部计算。此时，一个**"节点"**就是一个神经元。

[截图待插入: 左侧为详细神经元图（a1/a2/a3经w1/w2/w3到包含sum和sgn的圆圈，输出z）；右侧为简化节点图（a1/a2/a3经w1/w2/w3到Node圆圈，输出z）]

### MP神经元的理解

神经元模型可以这样理解：

- 根据3个**已知属性**预测**未知属性**
- 神经元模型中：3个已知属性的值是 $a_1$，$a_2$，$a_3$，未知属性的值是 $z$
- 已知的属性称为**特征**，未知属性称为**目标**

| 已知（特征） | 权值 | 计算过程 | 未知（目标） |
|-------------|------|----------|-------------|
| $a_1$ 细胞大小 | $w_1$ | 求和 sum → 激活函数 sgn | $z$ |
| $a_2$ 细胞形状 | $w_2$ | | 恶性肿瘤 / 良性肿瘤 |
| $a_3$ 分化程度 | $w_3$ | | |

### MP神经元的缺陷

1943年发布的MP模型，虽然简单，但已经建立了神经网络大厦的地基。然而，MP模型中，**权重的值都是预先设置的**，因此不能进行学习。

1949年心理学家Hebb发现人脑神经细胞的突触上的强度是可以变化的，提出了**Hebb学习率**。这启发计算机科学家们开始考虑用调整权值的方法来让**机器自我学习**，为后面的学习算法奠定了基础。

基于这个启发，才**诞生了感知机（Perceptron）**。

### 感知机模型

**感知机（Perceptron）**：由两层神经元构成

- **输入层**：在MP模型的输入位置添加神经元节点，**只传输数据**，不做计算
- **输出层**：对前面一层的输入**进行计算**

[截图待插入: 感知机结构图，输入层a1/a2/a3（红色节点）经权值w1,1/w1,2/w2,1/w2,2/w3,1/w3,2连接到输出层f（绿色节点），输出z1/z2]

感知机拥有一个计算层，也称为"单层神经网络"。

### 感知机模型数学化公式

输入向量 $\boldsymbol{a} = [a_1, a_2, a_3]^T$，输出向量 $\boldsymbol{z} = [z_1, z_2]^T$，系数矩阵 $\boldsymbol{W} \in \mathbb{R}^{2 \times 3}$。

则用矩阵乘法表达感知机的计算公式为：$\boldsymbol{z} = g(\boldsymbol{W} \times \boldsymbol{a})$。

该公式也是**神经网络中从前一层到后一层的运算过程**。

$$z_1 = g(a_1 w_{1,1} + a_2 w_{1,2} + a_3 w_{1,3})$$
$$z_2 = g(a_1 w_{2,1} + a_2 w_{2,2} + a_1 w_{2,3})$$

[截图待插入: 感知机网络图，输入a1/a2/a3经权值矩阵W连接到输出f节点，输出z1/z2]

### 感知机模型数学化公式（训练）

与神经元MP模型不同，感知机中的权重是通过**训练**得到的。

目标：$\min_{\boldsymbol{W}} |\boldsymbol{y} - \hat{\boldsymbol{y}}|$

[截图待插入: 感知机训练图，特征a1到an经权值w1/w2/wn连接到节点，输出 $\hat{y} = \text{sgn}(\boldsymbol{w} \times \boldsymbol{a})$，目标为y]

$$\text{sgn}(z) = \begin{cases} 1, & \text{if } z \geq 0 \\ -1, & \text{otherwise} \end{cases}$$

**感知机训练法则：**

$$W_t \leftarrow W_t + \Delta W_t$$
$$\Delta W_t = \eta (y - \hat{y}) \boldsymbol{a}$$

其中：
- $y$：是真实的目标
- $\hat{y}$：是感知机的输出
- $\eta$：学习速率（如 0.1）
- $\boldsymbol{a}$：训练数据

### 感知机的缺陷（实验结论）

**实验结论**：感知机只能做简单的线性分类任务。

- 给定感知机 $\boldsymbol{z} = \text{sgn}(\boldsymbol{W} \times \boldsymbol{a})$
- 考虑 $\boldsymbol{W} \times \boldsymbol{a} = 0$，即 $w_1 \cdot a_1 + w_2 \cdot a_2 = 0$ → 一条直线 → 决策平面
- 不在直线上的点 $\boldsymbol{a}$ 代入 $\boldsymbol{W} \times \boldsymbol{a}$，结果 $>0$ 或 $<0$ 经过sgn函数的作用就会划分出1或-1两个分类

### 多层感知机

单层神经网络无法解决异或问题。但是当增加一个计算层以后，两层神经网络不仅可以解决异或问题，而且具有非常好的非线性分类效果。不过两层神经网络的计算是一个问题，没有一个较好的解法。

1986年，Rumelhar和Hinton等人提出了反向传播（Backpropagation，BP）算法，解决了两层神经网络所需要的复杂计算量问题，从而带动了业界使用两层神经网络研究的热潮。

这时候的Hinton还很年轻，30年以后，正是他重新定义了神经网络，带来了神经网络复苏。

### 多层感知机模型（多层神经网络/MLP）

- **多层神经网络**：即**多层感知机（Multi-Layer Perceptron，简称MLP）**
- 包含三个层次：一个**输入层**，一个或多个中间层（也叫**隐藏层**，hidden layer）和一个**输出层**
- 输入层与输出层的节点数是固定的，中间层则可以自由指定

[截图待插入: MLP结构图，输入层（3个红色节点）→ 隐藏层（4个紫色节点）→ 输出层（2个绿色节点），全连接]

### 多层感知机模型数学化表达

使用向量和矩阵来表示神经网络中的变量。$\boldsymbol{x}$, $\boldsymbol{a}$, $\boldsymbol{z}$ 是网络中传输的向量数据。$\boldsymbol{W_1}$ 和 $\boldsymbol{W_2}$ 是网络的矩阵参数。

矩阵运算具有简洁性，节点数增多也不受影响。

$$\boldsymbol{a} = g(\boldsymbol{W_1} \times \boldsymbol{x})$$
$$\boldsymbol{z} = g(\boldsymbol{W_2} \times \boldsymbol{a}) = g(\boldsymbol{W_2} \times g(\boldsymbol{W_1} \times \boldsymbol{x}))$$

[截图待插入: 两层神经网络图，输入层x（3个红色节点）经W1到隐藏层a（2个紫色节点）经W2到输出层z（2个绿色节点）]

### 多层感知机模型数学化表达（偏置单元）

MLP通常还会引入**偏置单元**，它与后一层的所有节点都有连接。

记所有偏置值为向量 $\boldsymbol{b}$，则MLP的矩阵运算公式为：

$$\boldsymbol{a} = g_1(\boldsymbol{W_1} \times \boldsymbol{x} + \boldsymbol{b_1})$$
$$\boldsymbol{z} = g_2(\boldsymbol{W_2} \times \boldsymbol{a} + \boldsymbol{b_2}) = g_2(\boldsymbol{W_2} \times g_1(\boldsymbol{W_1} \times \boldsymbol{x} + \boldsymbol{b_1}) + \boldsymbol{b_2})$$

[截图待插入: 带偏置的MLP图，输入层x（含+1偏置节点b1）经W1到隐藏层a（含+1偏置节点b2）经W2到输出层z]

### 多层感知机模型激活函数

两层神经网络**不再使用sgn函数**作为激活函数，而是使用**平滑函数sigmoid**。

**不同的激活函数：**

| 函数 | 公式 | 说明 |
|------|------|------|
| (a) sgn | $\text{sgn}(x) = \begin{cases} 1, & x > 0 \\ -1, & x < 0 \end{cases}$ | 用于MP神经元，感知机 |
| (b) sigmoid | $\text{sigmoid}(x) = \frac{1}{1 + e^{-x}}$ | 多用于多层神经网络(MLP) |
| (c) ReLU | $\text{ReLU}(x) = \max(0, x)$ | 多用于深层神经网络(DNN) |
| (d) tanh | $\text{tanh}(x) = \frac{e^x - e^{-x}}{e^x + e^{-x}}$ | 多用于深层神经网络(DNN) |

[截图待插入: 四种激活函数图像对比，(a)sgn阶梯函数，(b)sigmoid S形曲线，(c)ReLU折线，(d)tanh S形曲线过原点]

### 多层神经网络的拟合原理

**通用近似定理（Universal Approximation Theorem）**[Cybenko, 1989, Hornik et al., 1989]：

令 $\varphi(\cdot)$ 是一个非常数、有界、单调递增的连续函数，$\mathbb{I}_d$ 是一个d维的单位超立方体 $[0,1]^d$，$C(\mathbb{I}_d)$ 是定义在 $\mathbb{I}_d$ 上的连续函数集合。对于任何一个函数 $\boldsymbol{f(x)} \in C(\mathbb{I}_d)$，存在一个整数 $m$，和一组实数 $v_i, b_i \in \mathbb{R}$ 以及实数向量 $\boldsymbol{W_i} \in \mathbb{R}^d, i = 1, \cdots, m$，可以定义函数：

$$F(\boldsymbol{x}) = \sum_{i=1}^{m} v_i \varphi(\boldsymbol{W_i^T x} + b_i)$$

作为函数 $f$ 的近似实现，即：

$$|F(\boldsymbol{x}) - f(\boldsymbol{x})| < \epsilon, \forall \boldsymbol{x} \in L_d \quad \text{其中} \epsilon > 0 \text{是一个很小的正数。}$$

[截图待插入: 神经网络拟合示意图，输入x经W1到隐藏层φ节点（z1-z4），再经v1-v4加权求和到Σ输出F(x)，偏置b1]

- 多层神经网络的本质就是通过**矩阵参数**与**激活函数**来拟合特征与目标之间的真实函数关系
- 因此，**多层神经网络也能够解决XOR异或问题！**

### 多层感知机效果（总结）

两层神经网络可以做非线性分类的关键--隐藏层。

矩阵公式中矩阵和向量相乘以及非线性函数sigmod/tanh/...，本质上就是对向量的坐标空间进行一个变换。因此，隐藏层的作用就是使得数据的原始坐标空间**从线性不可分，转换成了线性可分**。

两层神经网络通过两层的线性模型模拟了数据内真实的非线性函数。因此，多层的神经网络的**本质就是复杂函数拟合**。

设计一个神经网络时，输入层的节点数需要与特征的维度匹配，输出层的节点数要与目标的维度匹配。如何决定这个自由层的节点数呢？一般根据经验来设置。

**Grid Search（网格搜索）**：预先设定几个可选值，通过切换这几个值来看整个模型的预测效果，选择效果最好的值作为最终选择。
