## 建议复习顺序

1. 先复习搜索三讲：理解“问题如何形式化”和“算法如何保证完备/最优”。
2. 再复习不确定性推理两讲：概率公式、条件独立、贝叶斯网络推断。
3. 接着复习机器学习三讲：决策树、机器学习范式、聚类。
4. 最后复习神经网络三讲：MLP/BP、CNN/RNN/LSTM、Transformer。

## 高频公式

- 搜索评价函数：$f(n)=g(n)+h(n)$。
- A* 可采纳性：$h(n) \le h^*(n)$。
- A* 一致性：$h(n_1) \le c(n_1,n_2)+h(n_2)$。
- 贝叶斯法则：$P(Y|X)=P(X|Y)P(Y)/P(X)$。
- 朴素贝叶斯：$\arg\max_C P(C)\prod_i P(A_i|C)$。
- 熵：$H(X)=-\sum_x p(x)\log_2 p(x)$。
- 信息增益：$Gain(D,A)=H(D)-H(D|A)$。
- Gini：$gini(D)=1-\sum_j p_j^2$。
- K-means 复杂度：$O(nKId)$。
- 卷积输出尺寸：$h=(n_h-f+2p)/s+1$, $w=(n_w-f+2p)/s+1$。
- 自注意力：$Q=XW_Q$, $K=XW_K$, $V=XW_V$, $Att=softmax(QK^T/\sqrt{d_k})V$。
