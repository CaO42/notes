## 深度学习

### 多层感知机结构与数学化

多层感知机包含输入层、隐藏层（可多层）和输出层。设 $g$ 为激活函数，各层计算如下：

$$g(\boldsymbol{W}^{(1)} \times \boldsymbol{x}^{(1)}) = \boldsymbol{h}^{(1)}$$

$$g(\boldsymbol{W}^{(2)} \times \boldsymbol{h}^{(1)}) = \boldsymbol{h}^{(2)}$$

$$g(\boldsymbol{W}^{(3)} \times \boldsymbol{h}^{(2)}) = y'$$

整体可写为复合函数：

$$g\left(\boldsymbol{W}^{(3)} \times g\left(\boldsymbol{W}^{(2)} \times g(\boldsymbol{W}^{(1)} \times \boldsymbol{x}^{(1)})\right)\right) = y'$$

**核心思想**：神经网络通过**矩阵参数 $\boldsymbol{W}$** 与**激活函数 $g$** 来拟合特征与目标之间的真实关系。

### 多层感知机如何训练

以房价预测为例，输入特征包括大小 $x_1$、房间数 $x_2$、区域编码 $x_3$、收入水平 $x_4$，输出预测价格 $y'$，与真实价格 $y$ 对比。

$$y' = g\left(\boldsymbol{W}^{(3)} \times g\left(\boldsymbol{W}^{(2)} \times g(\boldsymbol{W}^{(1)} \times \boldsymbol{x}^{(1)})\right)\right)$$

**损失函数**：样本的预测结果为 $out$ ($y'$)，真实目标为 $target$ ($y$)。损失 $loss$ 的计算公式如下，目标是最小化预测结果与真实目标之间的距离：

$$Loss = \frac{1}{2}(y - y')^2$$

**应用案例**：我国崔铁军院士课题组基于多层感知器开发了一种基于高分辨率可编程超表面的**微波成像系统**。相比一般光学成像，微波成像系统具有高穿透性、抗视距障碍物、全天候等特征，广泛应用于安检、自动化检测和国防等领域。

### 训练多层感知机

- Rosenblat提出的感知机模型中，模型中的参数可以被训练，但是使用的方法较为简单，并没有使用目前机器学习中通用的方法，这导致其扩展性与适用性非常有限。
- 从两层神经网络开始，神经网络的研究人员开始使用机器学习相关的技术进行神经网络的训练。例如用大量的数据（1000-10000左右），使用算法进行优化等等，从而使得模型训练可以获得性能与数据利用上的双重优势。
- 机器学习模型训练的目的，就是使得参数尽可能的与真实的模型逼近。具体做法是这样的：首先给所有参数赋上随机值，然后使用这些随机生成的参数值，来预测训练数据中的样本，不断循环迭代更新参数。
- 样本的预测目标为 $out$，真实目标为 $target$。**损失** $loss$ 的计算公式如下，目标是使对所有训练数据的损失和尽可能的小：

$$loss = (target - out)^2$$

### 如何优化参数

**问题**：如何优化参数，能够让损失函数的值最小？

**前向传播**：按顺序（从输入层到输出层）计算和存储神经网络中每层的结果。目的在于**计算当前参数下的误差**。

**反向传播**：计算神经网络参数梯度的方法。根据微积分中的链式规则，按相反的顺序从输出层到输入层遍历网络。目的在于**根据误差调整参数**。

### 多层感知机如何训练：前向传播 + 反向传播（示例）

**示例**：一个具有两个输入、两个隐藏神经元和两个输出神经元的神经网络。

**基本结构和初始化**：

- 输入：$x_1 = 0.05$，$x_2 = 0.10$
- 目标输出：$target_{o1} = 0.01$，$target_{o2} = 0.99$
- 初始权重：$w_1=0.15$，$w_2=0.20$，$w_3=0.25$，$w_4=0.30$，$w_5=0.40$，$w_6=0.45$，$w_7=0.50$，$w_8=0.55$
- 偏置：$b_1=0.35$，$b_2=0.60$

**目标**：进行**优化权重**，使神经网络能够**学习如何正确地将任意输入映射到输出**。

### 前向传播：计算隐藏层

**计算 $h_1$ 的输入**：

$$net_{h_1} = w_1 \times x_1 + w_2 \times x_2 + b_1 \times 1 = 0.15 \times 0.05 + 0.2 \times 0.1 + 0.35 \times 1 = 0.3775$$

**使用激活函数（sigmoid）计算输出**：

$$out_{h_1} = \frac{1}{1 + e^{-net_{h_1}}} = 0.59326992$$

**对 $h_2$ 进行相同的计算**：

$$out_{h_2} = \frac{1}{1 + e^{-net_{h_2}}} = 0.596884378$$

### 前向传播：计算输出层

**计算 $o_1$ 的输入**：

$$net_{o_1} = w_5 \times h_1 + w_6 \times h_2 + b_2 \times 1 = 0.4 \times 0.59326992 + 0.45 \times 0.596884378 + 0.35 \times 1 = 1.105905967$$

**使用激活函数计算输出**：

$$out_{o1} = \frac{1}{1 + e^{-net_{o1}}} = 0.75136507$$

**计算 $o_2$ 的输出**：

$$out_{o2} = \frac{1}{1 + e^{-net_{o2}}} = 0.772928465$$

### 计算总误差

$$E_{total} = \frac{1}{2}\sum(target - output)^2$$

$$E_{o1} = \frac{1}{2}(target_{o1} - out_{o1})^2 = \frac{1}{2}(0.01 - 0.75136507)^2 = 0.274811083$$

$$E_{o2} = \frac{1}{2}(target_{o2} - out_{o2})^2 = \frac{1}{2}(0.99 - 0.772928465)^2 = 0.023560026$$

$$E_{total} = E_{o1} + E_{o2} = 0.298371109$$

**预测与目标的总误差！**

**问题**：如何根据总误差调整参数 $w$？

### 多层感知机如何训练：梯度下降

**回顾数学分析**：给定多元可微函数 $\mathcal{L}(\boldsymbol{w})$ 和其函数曲线上的一点 $P$，函数 $\mathcal{L}$ 在点 $P$ 上的**梯度**就是函数 $\mathcal{L}(\boldsymbol{w})$ 在点 $P$ 上的偏导数。

$$\mathcal{L}(\boldsymbol{w}) = \frac{1}{2}\sum_{n=1}^{N}(target - output(\boldsymbol{w}))^2$$

梯度：

$$\frac{\partial \mathcal{L}(w)}{\partial w} = \lim_{\Delta w \to 0}\frac{\mathcal{L}(w + \Delta w)}{\Delta w}$$

**路线**：找到梯度下降的方向，求取 $\mathcal{L}(\boldsymbol{w})$ 的最小值。

### 梯度下降法

梯度下降法：沿着梯度下降的方向更新参数，使得 $\mathcal{L}(\boldsymbol{w})$ 取值最小。

步骤：
1. 初始化 $\boldsymbol{w}$
2. 重复：
   - 计算梯度
   - 更新参数

参数更新公式：

$$\boldsymbol{w}^{[l]} \leftarrow \boldsymbol{w}^{[l]} - \alpha \frac{\partial \mathcal{L}(\boldsymbol{w})}{\partial \boldsymbol{w}^{[l]}}$$

### 反向传播算法

**反向传播算法**：**根据网络的特点而设计的高效梯度下降方法**，用于参数的学习。

复习前向传播中神经网络内部的计算：

$$g(\boldsymbol{W}^{(1)} \times \boldsymbol{x}^{(1)}) = \boldsymbol{h}^{(1)} \quad g(\boldsymbol{W}^{(2)} \times \boldsymbol{h}^{(1)}) = \boldsymbol{h}^{(2)}$$

$$g(\boldsymbol{W}^{(3)} \times \boldsymbol{h}^{(2)}) = y' \quad \mathcal{L} = \frac{1}{2}(y - y')^2$$

（粗体字表示向量，上标是层号）

这是关于 $\boldsymbol{W}^{(1)}$ 的**复合函数**，神经网络中应用**链式求导法则**进行梯度计算：

$$\frac{\partial \mathcal{L}}{\partial \boldsymbol{W}^{(1)}} = \frac{\partial \mathcal{L}}{\partial y'} \times \frac{\partial y'}{\partial \boldsymbol{h}^{(2)}} \times \frac{\partial \boldsymbol{h}^{(2)}}{\partial \boldsymbol{h}^{(1)}} \times \frac{\partial \boldsymbol{h}^{(1)}}{\partial \boldsymbol{W}^{(1)}}$$

### 多层感知机 - 训练

**梯度下降算法**：每次计算参数在当前的梯度，然后让参数向着梯度的反方向前进一段距离，不断重复，直到梯度接近零时截止。一般这个时候，所有的参数恰好达到使损失函数达到一个最低值的状态。

**反向传播算法**：反向传播算法不一次计算所有参数的梯度，而是从后往前。首先计算输出层的梯度，然后是第二个参数矩阵的梯度，接着是中间层的梯度，再然后是第一个参数矩阵的梯度，最后是输入层的梯度。计算结束以后，所要的两个参数矩阵的梯度就都有了。

### 反向传播：输出层

反向传播从输出层开始。总误差公式：

$$E_{total} = \sum \frac{1}{2}(target_{oi} - out_{oi})^2 = E_{o1} + E_{o2}$$

### 反向传播：考虑 $w_5$

探究 $w_5$ 的变化对总误差的影响有多大？使用**链式法则**：

$$\frac{\partial E_{total}}{\partial w_5} = \frac{\partial E_{total}}{\partial out_{o1}} \times \frac{\partial out_{o1}}{\partial net_{o1}} \times \frac{\partial net_{o1}}{\partial w_5}$$

**第一步**：计算 $\frac{\partial E_{total}}{\partial out_{o1}}$

$$E_{total} = \sum \frac{1}{2}(target_i - out_{oi})^2$$

$$\frac{\partial E_{total}}{\partial out_{o1}} = 2 \times \frac{1}{2}(target_{o1} - out_{o1})^{2-1} \times (-1) + 0 = -(target_{o1} - out_{o1}) = -(0.01 - 0.75136507) = 0.74136507$$

### 反向传播：Sigmoid 函数导数

**Sigmoid函数**：

$$out_{o1} = \sigma(x) = \frac{1}{1 + e^{-x}} \quad \frac{d}{dx}\sigma(x) = \sigma(x)(1 - \sigma(x))$$

$$\frac{\partial out_{o1}}{\partial net_{o1}} = out_{o1}(1 - out_{o1}) = 0.186815602$$

### 反向传播：计算 $\frac{\partial net_{o1}}{\partial w_5}$

$$net_{o1} = w_5 \times h_1 + w_6 \times h_2 + b_2 \times 1$$

$$\frac{\partial net_{o1}}{\partial w_5} = 1 \times h_1 \times w_5^{(1-1)} + 0 + 0 = h_1 = 0.593269992$$

### 反向传播：汇总并更新 $w_5$

$$\frac{\partial E_{total}}{\partial w_5} = -(target_{o1} - out_{o1}) \times out_{o1}(1 - out_{o1}) \times h_1$$

$$= 0.74136507 \times 0.186815602 \times 0.593269992 = 0.082167041$$

为了减少误差，设置**学习率 $\eta = 0.5$**：

$$w_5^+ = w_5 - \eta \times \frac{\partial E_{total}}{\partial w_5} = 0.4 - 0.5 \times 0.082167041 = 0.35891648$$

### 多层神经网络的问题

存在问题：尽管使用了BP算法，一次多层神经网络的**训练仍然耗时太久**，而且困扰训练优化的一个问题就是**局部最优解**问题，这使得神经网络的优化较为困难。同时，隐藏层的**节点数需要调参**，这使得使用不太方便，工程和研究人员对此多有抱怨

### 多层感知机激活函数的影响

**问题：为什么会产生这样的问题？**

问题：现实应用中，人们发现多层感知机不同的层的学习速度差异很大。

- 使用sigmoid作为神经元的输入输出函数，在神经网络反向传播梯度时，每传递一层梯度衰减为原来的0.25。层数一多，梯度指数衰减后低层基本上接受不到有效的训练信号。

2层：
$$\frac{\partial E_{total}}{\partial w_1} = \frac{\partial E_{total}}{\partial out_{h1}} * \frac{\partial out_{h1}}{\partial net_{h1}} * \frac{\partial net_{h1}}{\partial w_1}$$

$$\frac{\partial E_{total}}{\partial out_{h1}} = \frac{\partial E_{o1}}{\partial out_{h1}} + \frac{\partial E_{o2}}{\partial out_{h1}}$$

1层：
$$\frac{\partial E_{o1}}{\partial out_{h1}} = \frac{\partial E_{o1}}{\partial out_{o1}} * \frac{\partial out_{o1}}{\partial net_{o1}} * \frac{\partial net_{o1}}{\partial out_{h1}}$$

Sigmoid函数：$\sigma(x) = \frac{1}{1+e^{-x}}$

$$0 \leq \frac{d}{dx}\sigma(x) = \sigma(x)(1-\sigma(x)) \leq 0.25$$

[截图待插入: Sigmoid函数曲线图]

逐层递减直到**梯度消失**，使得多层网络等价于浅层网络的学习

### 多层训练解决手段

- **2006年Hinton提出的逐层预训练方法**
  - pre-training + fine-tunning...

[截图待插入: sigmoid与ReLU函数曲线对比图]

- **使用Rectified Linear Unit(ReLU)激活函数**
  - sigmoid等函数，算激活函数时（指数运算），计算量大，反向传播求误差梯度时，求导涉及除法，计算量相对大，而采用ReLU激活函数，整个过程的计算量节省很多
  - sigmoid函数反向传播时，容易出现梯度消失
  - ReLU会使一部分神经元的输出为0，这样就造成了网络的稀疏性，并且减少了参数的相互依存关系，缓解了过拟合问题的发生

- **Dropout**
  - 模型训练时随机让网络某些隐含层节点的权重不工作，防止过拟合

### 不同激活函数求导：ReLU详解

解决：使用Rectified Linear Unit(ReLU)激活函数

**Sigmoid函数及求导：**

$$\sigma(x) = \frac{1}{1+e^{-x}}$$

$$0 \leq \frac{d}{dx}\sigma(x) \leq 0.25$$

[截图待插入: sigmoid函数曲线图]

**ReLU函数及求导：**

$$\sigma(x) = \max(0, x)$$

$$\frac{d}{dx}\sigma(x) = \begin{cases} 0 & x \leq 0 \\ 1 & x > 0 \end{cases}$$

[截图待插入: ReLU函数曲线图]

- Sigmoid函数反向传播时，容易出现梯度消失，**ReLU函数避免梯度消失**
- Sigmoid等函数计算量大，反向传播时计算量相对大，采用ReLU激活函数的**计算量节省**很多
- ReLU会使部分神经元的输出为0，造成了网络的稀疏性，减少了参数的相互依存关系，**缓解过拟合问题**的发生

### 课堂练习

考虑以下神经网络，其中 node1 和 node2 为输入节点，node3 为输出节点，且输入节点均没有应用激活函数。输出节点 node3 的输入 $I_3 = w_{13} * x_1 + w_{23} * x_2 + \theta$，输出节点采用 sigmoid 激活函数，即 $O_3 = \frac{1}{1+e^{-I_3}}$，假定一个训练样本，$x_1 = 1, x_2 = 0.5$，其真实的类标签 $y = 1$，设损失函数采用均方误差，即 $L = 0.5 * (y - O_3)^2$，用以更新网络参数。当前网络的参数初始值为：$\theta = 0, w_{13} = 0.5, w_{23} = -1$。请基于上述训练样本的 $x_1, x_2, y$ 的取值，以及网络中 $\theta, w_{13}, w_{23}$ 的初始值，计算损失函数 $L$ 对 $w_{13}$ 的偏导，即 $\frac{\partial L}{\partial w_{13}}$ 的值（$\sqrt{e} = 1.65$）。

[截图待插入: 简单神经网络图，node1($x_1$)和node2($x_2$)通过权重$w_{13}$、$w_{23}$连接到node3，偏置$\theta$]

**解答：**

正向传播：

$$I_3 = w_{13} * x_1 + w_{23} * x_2 + \theta = 0$$

$$O_3 = \frac{1}{1+e^{-I_3}} = \frac{1}{2}$$

链式法则：

$$\frac{\partial L}{\partial w_{13}} = \frac{\partial L}{\partial O_3} * \frac{\partial O_3}{\partial I_3} * \frac{\partial I_3}{\partial w_{13}} = (O_3 - y) * \frac{e^{-I_3}}{(1+e^{-I_3})^2} * x_1 = -\frac{1}{2} * \frac{1}{4} * 1 = -\frac{1}{8}$$

### 特征图的维度

假设图片为（高度 $n_h$ x 宽度 $n_w$），卷积核为（$f \times f$），步长为 $s = 1$，填充 $p = 0$。

特征图为（高度 $h$ x 宽度 $w$）。

示例：输入 6x6，Filter 3x3，Stride $s=1$，Padding $p=0$。

第一个位置计算：

$$2 = 4 \times 1 + 9 \times 0 + 2 \times (-1) + 5 \times 1 + 6 \times 0 + 2 \times (-1) + 2 \times 1 + 4 \times 0 + 5 \times (-1)$$

### 特征图的维度（分组讨论：维度公式）

假设图片为（高度 $n_h$ x 宽度 $n_w$），卷积核为（$f \times f$），步长为 $s = 2$。

示例：输入 6x6，Filter 3x3，Stride $s=2$，Padding $p=0$。

第二个位置计算：

$$1 = 2 \times 1 + 5 \times 0 + 3 \times (-1) + 2 \times 1 + 4 \times 0 + 3 \times (-1) + 5 \times 1 + 4 \times 0 + 2 \times (-1)$$

### 特征图的维度（维度公式）

特征图维度公式：

$$高度\ h = \left\lfloor \frac{n_h - f}{s} + 1 \right\rfloor$$

$$宽度\ w = \left\lfloor \frac{n_w - f}{s} + 1 \right\rfloor$$

### 特征图的维度（含填充 padding）

假设图片为（高度 $n_h$ x 宽度 $n_w$），卷积核为（$f \times f$），步长为 $s$，填充宽度为 $p$。

特征图维度公式：

$$高度\ h = \left\lfloor \frac{n_h - f + 2p}{s} + 1 \right\rfloor$$

$$宽度\ w = \left\lfloor \frac{n_w - f + 2p}{s} + 1 \right\rfloor$$

示例：输入 6x6，Padding $p=1$（外围补零），Filter 3x3，Stride $s=2$。

第一个位置计算（含 padding 的零值）：

$$-15 = 0 \times 1 + 0 \times 0 + 0 \times (-1) + 0 \times 1 + 4 \times 0 + 9 \times (-1) + 0 \times 1 + 9 \times 0 + 6 \times (-1) = -15$$

### 特征图的维度（多通道输入）

图片为（高度 $n_h$ x 宽度 $n_w$ x 深度 $n_c$），卷积核为（$f \times f \times n_c$），步长为 $s$，填充为 $p$。

特征图维度公式：

$$高度\ h = \left\lfloor \frac{n_h - f + 2p}{s} + 1 \right\rfloor$$

$$宽度\ w = \left\lfloor \frac{n_w - f + 2p}{s} + 1 \right\rfloor$$

示例：输入 $n_H \times n_W \times n_C = 6 \times 6 \times 3$，Filter 3x3x3，Stride $s=1$，Padding $p=0$。

每个通道分别做卷积后求和：$2 = \text{红色通道卷积结果} + \text{绿色通道卷积结果} + \text{蓝色通道卷积结果}$

### 特征图的维度（K 个卷积核）

图片为（高度 $n_h$ x 宽度 $n_w$ x 深度 $n_c$），$K$ 个卷积核为（$f \times f \times n_c$），步长为 $s$，填充为 $p$。

特征图维度公式：

$$高度\ h = \left\lfloor \frac{n_h - f + 2p}{s} + 1 \right\rfloor$$

$$宽度\ w = \left\lfloor \frac{n_w - f + 2p}{s} + 1 \right\rfloor$$

$$深度\ k = K$$

示例：输入 6x6x3，Filter 1 为 3x3x3，Filter 2 为 3x3x3，各产生 4x4 特征图，堆叠后输出为 4x4x2。

### 循环神经网络（全连接网络的问题）

**使用全连接神经网络解决？**

对句子 "arrive Guangzhou on May 31th" 逐词标注：other, dest, other, time, time。

对句子 "leave Guangzhou on May 31th"，"Guangzhou" 应标注为 place of departure（出发地）而非目的地。

> 存在什么问题？

> **神经网络需要记忆功能！**

全连接网络将 "Guangzhou" 作为独立输入 $x_1, x_2$，无法利用上下文信息（如前面的 "arrive" 或 "leave"）来区分同一词的不同含义。
