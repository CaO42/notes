## 一句话总结

本讲介绍深度学习中的典型网络结构：CNN 用于视觉特征提取，RNN/LSTM 用于序列建模，并补充机器学习可解释性的基本思想。

## 深度学习视角

深度神经网络可理解为多层特征提取器加线性分类器。层数增加使模型能学习更复杂的表示。

典型网络：

- 视觉数据: CNN、ResNet、DenseNet。
- 文本和序列: RNN、LSTM、Transformer、BERT、ChatGPT。
- 其他: AlphaFold 等。

## CNN 的动机

直接用 MLP 处理图像会导致参数量巨大，并忽略图像结构。人类视觉启发了 CNN：

- 局部特征很重要。
- 同一特征可出现在不同位置。
- 下采样压缩通常不改变对象识别。

对应设计：

- 局部连接。
- 卷积核权重共享。
- 池化下采样。

## 卷积运算

卷积核是可移动的权重矩阵，可在图像上滑动检测局部模式。

输出尺寸公式：

若输入高宽为 $n_h,n_w$，卷积核大小 $f$，步长 $s$，填充 $p$：

$h=(n_h-f+2p)/s+1$

$w=(n_w-f+2p)/s+1$

若输入有深度 $n_c$，每个卷积核尺寸为 $f\\times f\\times n_c$。若有 $K$ 个卷积核，输出深度为 $K$。

## 池化

最大池化 Max Pooling 取局部窗口最大值，用于压缩特征图，增强一定平移不变性，减少参数和计算。

典型 CNN 结构：

Convolution -> Activation -> Pooling -> Convolution -> Pooling -> Flatten -> Fully Connected -> Output

## CNN 与 ANN 的关系

从 ANN 视角看，CNN 是带有局部连接和参数共享约束的神经网络。它减少参数量，并让模型更适合图像局部结构。

应用：图像处理、医学图像分析、机器人视觉、高速公路测速等。

## RNN 的动机

普通全连接网络难以处理序列，因为每个输入词需要结合上下文。例如订票系统中，“arrive Guangzhou on May 31th”需要记住前文才能判断 Guangzhou 是目的地还是出发地。

RNN 引入隐藏状态作为记忆：

- 当前输入影响当前隐藏状态。
- 上一步隐藏状态也影响当前隐藏状态。
- 同一组参数在时间步之间共享。

## RNN 训练与问题

RNN 用 Backpropagation Through Time (BPTT) 训练，把时间展开后做反向传播。

问题：

- 长序列中梯度经过多次矩阵乘法，可能指数衰减或爆炸。
- tanh 导数有界，容易梯度消失。
- 误差表面可能非常平坦或陡峭。

## LSTM

LSTM 用记忆单元和门控机制缓解长依赖问题。

核心组件：

- Memory Cell: 保存长期信息。
- Input Gate: 控制写入。
- Forget Gate: 控制遗忘或重置。
- Output Gate: 控制输出。

LSTM 的代价是参数量约为普通 RNN 的数倍，但能更好处理长期依赖。

## 可解释机器学习

为什么需要可解释性：

- 高风险场景需要理由，如金融贷款、司法假释、医疗诊断。
- 帮助发现模型是否学到了错误偏见，如性别、肤色等。
- 便于模型诊断与改进。

两类解释：

- Local explanation: 解释某一次决策为什么这样，例如图像哪些区域导致“猫”的判断。
- Global explanation: 解释整个模型学到了什么，例如模型心中的“猫”长什么样。

基本思想：移除或扰动输入组件，观察输出变化。变化越大，说明该组件越重要。

可解释性与模型能力存在张力：线性模型更可解释但表达能力弱，深度模型强但更黑盒。目标不是完全理解模型内部，而是让模型更可控、更可信。
