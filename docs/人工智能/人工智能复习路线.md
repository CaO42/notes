# 人工智能课程复习路线

> 资料来源：当前目录下 12 份章节 Markdown，以及 `人工智能复习课.pdf`。这份路线按”先会做题，再补概念”的思路组织，适合期末前集中复习。

## 0. 总体判断

这门课不是单纯背概念，核心是三类能力：

1. **形式化与证明**：把自然语言转成命题/谓词逻辑，化子句集，用归结、合一、反证推出结论。
2. **算法过程与计算**：手推搜索树、A*、Minimax、Alpha-Beta、模拟退火、遗传算法、朴素贝叶斯、贝叶斯网络、ID3/CART、K-means、BP、CNN、RL 更新公式。
3. **概念比较**：符号主义/连接主义/行为主义，BFS/DFS/UCS/IDS，A* 可采纳性/一致性，MC/DP/TD，SARSA/Q-learning，DQN/策略梯度等。

建议复习时每章都产出三样东西：

- 一页公式/流程卡片。
- 至少一道完整手算题或证明题。
- 一个“容易混淆点”列表。

### Day 3：归结推理专题

阅读：

必须掌握：

### Day 4：盲目搜索 + A*

阅读：
- `03_搜索技术I.md` 搜索定义、BFS/DFS/UCS/DLS/IDS、启发式搜索、A*。
- `04_搜索技术II.md` 中 8 数码、传教士和野蛮人问题。

必须掌握：
- 搜索问题五要素：初始状态、动作、后继、目标测试、路径代价。
- BFS/DFS/UCS/DLS/IDS 的完备性、最优性、时间/空间复杂度。
- A*：`f(n)=g(n)+h(n)`。
- 可采纳性：`h(n) <= h*(n)`。
- 一致性/单调性：`h(n1) <= c(n1,n2)+h(n2)`。
- 一致性推出可采纳性；有环检测时，一致性更关键。
- 松弛问题、错位方块数、曼哈顿距离、启发式支配。

刷题：
- Water Jug 的 BFS/DFS 边界变化。
- 8 数码 A* 展开。
- 判断启发式是否可采纳/一致。

产出：
- 默写盲目搜索总结表。
- 写出 A* 最优性的条件和“环检测影响”的一句话解释。

### Day 5：对抗搜索 + 局部搜索 + 模拟退火

阅读：
- `04_搜索技术II.md` Minimax、Alpha-Beta、爬山法。
- `05_搜索优化算法.md` 模拟退火详细讲解。

必须掌握：
- Minimax 自底向上回传值。
- Alpha-Beta：Max 节点维护 alpha，Min 节点维护 beta；`alpha >= beta` 时剪枝。
- 最优排序下 Alpha-Beta 可从 `O(b^d)` 降到约 `O(b^(d/2))`。
- 评价函数：终止节点排序一致、计算快、与胜率强相关。
- 爬山法局限：局部最优、平台、山脊。
- 模拟退火接受概率：`min(1, exp(-ΔC/t))`，三函数两准则一初温。

刷题：
- Alpha-Beta 剪枝练习，必须能标出剪枝节点和最优路径。
- 模拟退火背包案例，按初始化、邻域、接受概率、降温流程复述。

产出：
- 一页“Minimax/Alpha-Beta 手推规则”。
- 一页“SA 伪代码 + 参数含义”。

### Day 6：遗传算法 + 贝叶斯推断 + 朴素贝叶斯

阅读：
- `05_搜索优化算法.md` 遗传算法部分。
- `05_不确定性知识表示与推理.md` 贝叶斯推断。
- `06_贝叶斯分类与贝叶斯网络.md` 朴素贝叶斯部分。

必须掌握：
- GA 六步：编码、初代种群、适应度、选择、交叉、变异。
- 选择概率：`p_i = f_i / Σf_j`。
- 贝叶斯公式、全概率公式、先验/后验/似然。
- 朴素贝叶斯分类：最大化 `P(C_j) Π_i P(A_i|C_j)`。
- 连续属性可用正态分布概率密度估计。
- 平滑化处理零概率。

刷题：
- 流感阳性后验概率例题。

### Day 7：贝叶斯网络 + D-Separation + 变量消除

阅读：
- `06_贝叶斯分类与贝叶斯网络.md` 贝叶斯网络、D-separation、BN inference、Variable Elimination。

必须掌握：
- 贝叶斯网络 = DAG + CPT。
- 联合分布分解：`P(X)=Π_i P(X_i | Parents(X_i))`。
- 三种结构：
  - Chain：`X -> Y -> Z`，给定中间节点阻断。
  - Fork：`X <- Y -> Z`，给定共同原因阻断。
  - Collider：`X -> Y <- Z`，默认阻断；给定碰撞点或其后代会打开。
- 推断：枚举法、变量消除。
- 因子 Join、Eliminate、Evidence、Normalize。

刷题：
- Burglary 网络联合概率计算。
- D-separation 判断独立性。
- Traffic 网络变量消除。

产出：
- 写一个变量消除模板：
  `实例化 evidence -> 选择隐藏变量 -> Join 含该变量因子 -> Sum out -> 合并剩余因子 -> Normalize`。

### Day 8：决策树 + 聚类 + 神经网络基础

阅读：
- `07_机器学习与决策树.md` 决策树 ID3/CART、机器学习三步骤、K-means/DBSCAN。
- `08_人工神经网络.md` MP/感知机/MLP。

必须掌握：
- 熵、条件熵、信息增益：`g(D,A)=H(D)-H(D|A)`。
- Gini：`1-Σp_j^2`，`Gini_split` 加权平均，选择最小者。
- 机器学习三步骤：Model、Loss、Optimization。
- K-means：分配点 -> 更新中心 -> 直到中心不变；SSE 评估；K-means++ 初始中心分散。
- DBSCAN：核心点、边界点、噪声点，Eps 与 MinPts。
- MP 神经元、感知机只能线性分类，XOR 需要多层感知机。

刷题：
- ID3 信息增益计算。
- CART Gini 计算。
- K-means 一次迭代。
- DBSCAN 点类型判断。

产出：
- 一张“ID3 vs CART vs Naive Bayes”对比表。

### Day 9：深度学习 + CNN + 序列模型/Transformer

阅读：
- `09_深度学习.md`
- `10_深度学习II.md`：RNN、LSTM、Word2Vec、Seq2Seq、Attention、Transformer。

必须掌握：
- 前向传播、损失函数、梯度下降更新。
- BP：链式法则，从输出层向输入层传梯度。
- Sigmoid 导数：`σ(x)(1-σ(x))`。
- ReLU 优点：缓解梯度消失、计算更简单、稀疏性。
- CNN 输出尺寸：
  `h=floor((n_h - f + 2p)/s + 1)`，`w=floor((n_w - f + 2p)/s + 1)`，K 个卷积核输出深度 K。
- Pooling 的作用：降维、保留局部显著特征、增强一定平移鲁棒性。
- RNN 处理序列，LSTM 用门控缓解长期依赖问题。
- Word2Vec：Skip-gram 与 CBOW。
- Attention：根据 query 对 values 加权求和；Transformer 用 self-attention 建模上下文。

刷题：
- BP 课堂练习中 `∂L/∂w13`。
- CNN 输出维度练习：`8 x 8 x 3` 输入、4 个 `3 x 3 x 3` 滤波器。
- 简述 Attention 相比普通 Seq2Seq 的优势。

产出：
- 一页“BP 链式法则手算模板”。
- 一页“CNN 尺寸公式 + 常见坑”。

### Day 10：强化学习专题 + 全局串联

阅读：
- `11_强化学习.md`

必须掌握：
- MDP 五元组：`<S,A,p,R,γ>`。
- 回报：`G_t = Σ γ^k R_{t+k+1}`。
- 状态价值 `Vπ(s)`、动作价值 `Qπ(s,a)`。
- Bellman 方程和 Bellman 最优方程。
- DP：策略评估、策略提升、策略迭代、值迭代。
- MC：完整 episode 后估计，方差大但不需要模型。
- TD：`V(s) <- V(s)+α(R+γV(s')-V(s))`。
- SARSA：on-policy，更新目标含实际选择的 `A_{t+1}`。
- Q-learning：off-policy，更新目标含 `max_a Q(S_{t+1},a)`。
- DQN：用神经网络近似 Q，经验回放池打破相关性，目标网络稳定训练。
- 策略梯度/REINFORCE/Actor-Critic：知道基本思想和区别。

刷题：
- MDP 轨迹回报计算。
- Bellman 方程列式。
- TD/SARSA/Q-learning 更新公式对比。

产出：
- 一张“DP/MC/TD/SARSA/Q-learning/DQN”对比表。
- 最后用 2 小时回看所有错题和卡片。

## 5. 高频混淆点

| 混淆点 | 正确区分 |
|---|---|
| 可采纳 vs 一致 | 可采纳是不高估到目标真实代价；一致是相邻节点满足三角不等式。一致推出可采纳。 |
| Graph search 环检测下 A* 最优性 | 只有可采纳不一定够；一致性能保证第一次扩展某状态就是最小代价路径。 |
| 自然演绎 vs 归结 | 自然演绎按推理规则直接推结论；归结通常用反证，把 `KB ∧ ¬α` 推到空子句。 |
| 合一 vs 置换 | 置换是变量替换集合；合一是找到置换让表达式一致；MGU 是最一般的合一。 |
| 枚举法 vs 变量消除 | 枚举先形成大联合再求和；变量消除边 join 边 sum out，避免中间因子过大。 |
| Chain/Fork/Collider | Chain/Fork 给定中间节点会阻断；Collider 默认阻断，给定碰撞点或后代会打开。 |
| ID3 vs CART | ID3 选信息增益最大；CART 选 Gini_split 最小。 |
| SARSA vs Q-learning | SARSA 用实际下一动作，on-policy；Q-learning 用下一状态最大动作值，off-policy。 |
| MC vs TD | MC 等 episode 结束用完整回报；TD 用一步奖励和下一状态估计即时更新。 |
| RNN vs LSTM vs Transformer | RNN 循环传隐藏状态；LSTM 加门控记忆；Transformer 用 self-attention 并行建模上下文。 |

## 6. 每轮复习的检查标准

第一轮完成时：
- 能说出每章在考什么。
- 每章至少有一页公式/流程卡片。

第二轮完成时：
- 归结、A*、Alpha-Beta、朴素贝叶斯、变量消除、ID3/CART、BP、CNN、RL 更新都至少手算过一次。
- 搜索和强化学习的公式能默写。

考前最后一轮：
- 只看错题、公式卡片和高频混淆点。
- 绪论、三大学派、AI 发展线、生成式 AI 对比放最后背。
- 所有手算题都按“写公式 -> 代数值 -> 得结论”的格式练，避免只会口头解释。
