导论：马克思主义中国化时代化的历史进程与理论成果



**如何理解毛泽东思想和中国特色社会主义理论体系是一脉相承的关系？ (2次)**

- 二者都坚持马克思主义基本立场、观点、方法，都是马克思主义中国化时代化的重要理论成果。
- 毛泽东思想是第一次历史性飞跃，为中国特色社会主义理论体系形成奠定理论基础、实践基础和制度基础。
- 中国特色社会主义理论体系继承并发展毛泽东思想，围绕改革开放和社会主义现代化建设回答新的时代课题。
- 二者既一脉相承，又与时俱进，统一于中国共产党推进民族独立、人民解放和国家富强、人民幸福的实践。

第一章：毛泽东思想及其历史地位



**毛泽东思想活的灵魂是什么？如何理解？ (6次)**

- 毛泽东思想活的灵魂是贯穿其中的立场、观点和方法，包括实事求是、群众路线、独立自主。
- 实事求是：一切从实际出发，理论联系实际，在实践中检验和发展真理，是思想路线的核心。
- 群众路线：一切为了群众，一切依靠群众，从群众中来，到群众中去，是党的生命线和根本工作路线。
- 独立自主：坚持独立思考，走自己的路，维护民族独立和国家主权，把立足点放在依靠自己力量上。
- 三者统一于中国革命、建设实践，是正确认识国情、依靠人民、探索中国道路的根本方法。



**独立自主的内涵和意义是什么？ (2次)**

- 独立自主，就是坚持独立思考，走自己的路。
- 对内要把中国事情放在依靠中国人民自己力量的基础上，坚定维护民族独立和国家主权。
- 对外要坚持独立自主的和平外交政策，同时积极争取外援，学习外国一切有益经验。
- 独立自主是中华民族精神之魂，是党和国家立党立国的重要原则。
- 意义：只有坚持独立自主，才能从中国实际出发解决中国问题，维护国家主权安全发展利益，在国际交往中保持主动。



**为什么说毛泽东思想是党和人民宝贵的精神财富？ (1次)**

- 毛泽东思想是马克思主义中国化时代化的第一个重大理论成果，是中国革命和建设的科学指南。
- 它指导中国革命取得胜利、建立新中国，并指导社会主义基本制度确立。
- 它为探索社会主义建设道路、开创中国特色社会主义提供了宝贵经验、理论准备和物质基础。
- 其中的实事求是、群众路线、独立自主和艰苦奋斗精神，仍是党和人民不断前进的重要精神动力。

本章简答、辨析、论述范围题



**实事求是的内涵是什么？**

- 实事求是，就是一切从实际出发，理论联系实际，坚持在实践中检验真理和发展真理。
- “实事”就是客观存在着的一切事物，“是”就是客观事物的内部联系即规律性，“求”就是研究。
- 坚持实事求是，要从中国基本国情出发，把握客观规律，反对主观主义和教条主义。
- 实事求是是毛泽东思想的精髓，也是党的思想路线的核心。



**毛泽东思想形成和发展的社会历史条件是什么？**

- 时代条件：世界进入帝国主义和无产阶级革命时代，俄国十月革命给中国送来马克思列宁主义。
- 理论基础：马克思列宁主义是毛泽东思想形成的理论基础。
- 国情依据：近代中国半殖民地半封建社会和中国革命的特殊实际，是理论形成的现实依据。
- 实践基础：中国共产党领导人民进行革命和建设的成功实践，以及正反两方面经验总结。
- 主观条件：以毛泽东同志为主要代表的中国共产党人，把马克思主义基本原理同中国具体实际相结合。



**毛泽东思想的主要内容有哪些？**

- 新民主主义革命理论，是毛泽东思想的基石。
- 社会主义革命和社会主义建设理论，解决中国由新民主主义向社会主义过渡和建设社会主义的问题。
- 革命军队建设和军事战略理论，强调党对军队绝对领导和人民战争思想。
- 政策和策略理论，强调原则性与灵活性相结合。
- 还包括思想政治工作和文化工作理论、党的建设理论等。

第二章：新民主主义革命理论



**如何理解新民主主义革命的三大法宝及其相互关系？ (2次)**

- 三大法宝是统一战线、武装斗争、党的建设。
- 统一战线是团结一切可以团结的力量，孤立主要敌人，是革命胜利的重要条件。
- 武装斗争是中国革命的主要斗争形式，是夺取政权的基本手段。
- 党的建设是掌握统一战线和武装斗争的根本保证。
- 三者关系：统一战线和武装斗争是战胜敌人的两个基本武器，党的组织则是掌握这两个武器的英勇战士。



**新民主主义革命的性质是什么？ (1次)**

- 近代中国半殖民地半封建社会性质和革命任务，决定它不是社会主义革命，而是资产阶级民主主义革命。
- 它又不是旧式民主主义革命，而是无产阶级领导的、人民大众的、反帝反封建反官僚资本主义的新民主主义革命。
- 它属于世界无产阶级社会主义革命的一部分，前途是经过新民主主义走向社会主义。
- 根本区别在于领导权掌握在无产阶级及其政党手中。



**新民主主义革命的总路线是什么？ (1次)**

- 总路线是：无产阶级领导的，人民大众的，反对帝国主义、封建主义和官僚资本主义的革命。
- 它指明了革命对象：帝国主义、封建主义、官僚资本主义。
- 它指明了革命动力：无产阶级、农民阶级、城市小资产阶级和民族资产阶级。
- 它指明了领导力量、革命性质和前途：无产阶级领导的新民主主义革命，前途是社会主义。

本章简答、辨析、论述范围题



**新民主主义基本纲领的主要内容是什么？**

- 政治纲领：推翻帝国主义和封建主义统治，建立无产阶级领导的、各革命阶级联合专政的新民主主义共和国。
- 经济纲领：没收封建地主阶级土地归农民所有，没收官僚资本归新民主主义国家所有，保护民族工商业。
- 文化纲领：发展无产阶级领导的人民大众的反帝反封建文化，即民族的科学的大众的文化。
- 三个纲领分别从政治、经济、文化上具体展开了新民主主义革命总路线。



**如何认识中国革命走农村包围城市、武装夺取政权道路的必要性及重大意义？**

- 必要性：近代中国内部没有资产阶级民主制度，外部受帝国主义压迫，革命只能以武装斗争为主要形式。
- 必要性：敌强我弱、城市反动统治力量强大，革命力量必须先在农村保存和发展。
- 可能性：中国政治经济发展不平衡，农村有广阔回旋余地，农民是革命主力军。
- 意义：这条道路把马克思主义基本原理同中国革命实际结合起来，开辟了中国革命胜利的正确道路。
- 意义：它丰富发展了马克思主义关于无产阶级革命道路的理论。



**新民主主义革命统一战线的必要性是什么？**

- 中国革命敌人异常强大，必须团结一切可以团结的力量，才能孤立和打击主要敌人。
- 近代中国社会矛盾复杂，革命动力包括工人、农民、城市小资产阶级和民族资产阶级，需要统一战线把他们联合起来。
- 中国革命长期性、残酷性和不平衡性，决定党必须善于争取同盟者，扩大革命力量。
- 统一战线是无产阶级政党实现领导权的重要条件，必须坚持独立自主、又联合又斗争的原则。

第三章：社会主义改造理论



**党在过渡时期的总路线是什么？ (3次)**

- 总路线概括为“一化三改”。
- “一化”是逐步实现国家的社会主义工业化。
- “三改”是逐步实现对农业、手工业和资本主义工商业的社会主义改造。
- 一化是主体，三改是两翼，二者相互联系、相互促进。
- 实质是在发展生产力的同时变革生产关系，把新民主主义社会逐步过渡到社会主义社会。

本章简答、辨析、论述范围题



**新民主主义社会为什么是一个过渡性社会？**

- 新民主主义社会不是独立社会形态，而是由新民主主义向社会主义转变的过渡社会。
- 经济上存在社会主义国营经济、个体经济、私人资本主义经济等多种经济成分。
- 政治上实行工人阶级领导的、以工农联盟为基础的人民民主专政。
- 社会主要矛盾逐步转变为工人阶级和资产阶级、社会主义道路和资本主义道路的矛盾。
- 其发展方向是经过社会主义改造进入社会主义社会。



**如何认识我国社会主义改造的基本经验？**

- 坚持社会主义工业化建设与社会主义改造同时并举。
- 采取积极引导、逐步过渡的方式，避免急躁冒进。
- 用和平方法进行改造，特别是对资本主义工商业实行赎买政策。
- 把对所有制的改造同对人的改造结合起来。
- 坚持党的领导，依靠群众，创造适合中国特点的社会主义改造道路。



**如何辨析“农业社会主义改造要遵循自愿互利原则”？**

- 该说法正确。农业社会主义改造必须坚持自愿互利、典型示范和国家帮助的原则。
- 自愿互利，就是尊重农民意愿，不能用强迫命令方法推进合作化。
- 典型示范，就是通过办好先进典型，让农民看到合作化优越性。
- 国家帮助，就是在政策、资金、技术、物资等方面支持农业合作化。
- 这些原则保证了农业改造比较平稳地推进，适合中国农村实际。



**如何理解中国确立社会主义基本制度的重大意义？**

- 标志着我国进入社会主义社会，实现了中国历史上最深刻最伟大的社会变革。
- 为当代中国一切发展进步奠定了根本政治前提和制度基础。
- 使广大劳动人民真正成为国家和社会的主人。
- 极大促进了社会生产力发展，为社会主义现代化建设创造制度条件。
- 丰富和发展了科学社会主义理论和实践。

第四章：社会主义建设道路初步探索的理论成果



**党对社会主义建设道路初步探索有哪些经验教训？ (2次)**

- 必须把马克思主义基本原理同中国实际相结合，探索符合中国特点的社会主义建设道路。
- 必须正确认识社会主义社会主要矛盾和根本任务，集中力量发展生产力。
- 必须从实际出发，建设规模和速度要同国力相适应，不能急于求成。
- 必须发展社会主义民主，健全社会主义法治。
- 必须坚持党的民主集中制和集体领导制度，加强执政党建设。
- 必须坚持对外开放，借鉴和吸收人类文明成果。



**为什么新中国成立后要把工业化作为经济建设的主要任务？ (1次)**

- 新中国成立时工业基础薄弱，改变落后面貌必须推进工业化。
- 工业化是巩固民族独立、增强国防实力、建立完整国民经济体系的需要。
- 社会主义制度的巩固和发展，需要强大的现代工业作为物质技术基础。
- 只有发展工业和生产力，才能逐步改善人民生活，实现国家富强。



**如何理解社会主义建设时期工业化发展道路思想？ (1次)**

- 工业化是社会主义建设的中心任务之一，重点是建立独立的、比较完整的工业体系和国民经济体系。
- 要以农业为基础、工业为主导，正确处理重工业、轻工业和农业的关系。
- 要坚持独立自主、自力更生，同时学习外国有益经验。
- 要从中国实际出发，统筹速度、比例、效益和人民生活，避免急于求成。



**社会主义建设道路初步探索的意义是什么？**

- 巩固和发展了我国社会主义制度，增强了人民走社会主义道路的信心。
- 为开创中国特色社会主义提供了宝贵经验、理论准备和物质基础。
- 丰富了科学社会主义的理论和实践。
- 形成的许多正确思想，仍对今天推进社会主义现代化建设具有启示意义。



**如何理解社会主义社会的基本矛盾？ (1次)**

- 社会主义社会仍然存在生产力和生产关系、经济基础和上层建筑之间的矛盾。
- 这种矛盾同旧社会根本不同，主要是非对抗性的，可以通过社会主义制度自身调整解决。
- 解决基本矛盾的根本途径，是不断解放和发展生产力，改革不适应生产力发展的体制机制。
- 要正确区分敌我矛盾和人民内部矛盾，人民内部矛盾主要用民主方法解决。

本章简答、辨析、论述范围题



**中共八大对我国社会矛盾作出了怎样的判断？**

- 中共八大指出，社会主义改造基本完成后，我国国内主要矛盾已经发生变化。
- 主要矛盾不再是工人阶级和资产阶级的矛盾，而是人民对于经济文化迅速发展的需要同当前经济文化不能满足人民需要的状况之间的矛盾。
- 党和人民当前的主要任务，是集中力量发展社会生产力，把我国尽快从落后的农业国变为先进的工业国。
- 这一判断为探索适合中国情况的社会主义建设道路提供了重要理论依据。



**党在社会主义建设道路初步探索中取得了哪些重要理论成果？**

- 提出调动一切积极因素为社会主义事业服务的思想。
- 提出正确认识和处理社会主义社会矛盾的思想。
- 提出走中国工业化道路的思想。
- 形成关于社会主义发展阶段、现代化建设战略目标、国防建设、党的建设等重要思想。
- 这些成果体现了党开始独立探索适合中国情况的社会主义建设道路。

第五章：中国特色社会主义理论体系的形成发展

本章简答、辨析、论述范围题



**中国特色社会主义理论体系形成发展的社会历史条件是什么？**

- 国际背景：和平与发展成为时代主题，世界多极化、经济全球化深入发展。
- 历史根据：社会主义建设正反两方面经验，特别是改革开放前后的实践总结。
- 现实依据：我国仍处于社会主义初级阶段，发展中出现一系列新情况新问题。
- 实践基础：改革开放和社会主义现代化建设的伟大实践。
- 理论条件：坚持把马克思主义基本原理同中国具体实际和时代特征相结合。



**如何把握中国特色社会主义理论体系形成发展过程？**

- 邓小平理论回答什么是社会主义、怎样建设社会主义，开创中国特色社会主义。
- “三个代表”重要思想回答建设什么样的党、怎样建设党，推动中国特色社会主义跨世纪发展。
- 科学发展观回答实现什么样的发展、怎样发展，推进中国特色社会主义新发展。
- 这一过程体现了马克思主义中国化时代化一脉相承、与时俱进，是改革开放以来党的理论创新成果。

第六章：邓小平理论



**邓小平理论的精髓是什么？**

- 邓小平理论的精髓是解放思想、实事求是。
- 解放思想，就是破除僵化观念，使思想和实际相符合。
- 实事求是，就是一切从社会主义初级阶段实际出发，在实践中检验真理和发展真理。
- 它推动了拨乱反正、改革开放和社会主义现代化建设。



**“三个有利于”标准是什么？ (2次)**

- 判断改革和各方面工作是非得失，归根到底看“三个有利于”。
- 是否有利于发展社会主义社会的生产力。
- 是否有利于增强社会主义国家的综合国力。
- 是否有利于提高人民生活水平。
- 其中发展生产力是首要的、最根本的标准。



**“两手抓，两手都要硬”的内涵是什么？ (1次)**

- 社会主义现代化建设要坚持全面发展，不能只抓一个方面。
- 核心是物质文明和精神文明两手抓，两手都要硬。
- 还包括一手抓改革开放，一手抓打击犯罪；一手抓民主建设，一手抓法制建设等。
- 目的在于保证经济发展、政治稳定、社会进步和人的全面发展相统一。



**邓小平反复思考的首要基本理论问题是什么？ (1次)**

- 首要基本理论问题是：什么是社会主义、怎样建设社会主义。
- 回答这个问题，关键是搞清社会主义本质和社会主义根本任务。
- 社会主义本质是解放生产力、发展生产力，消灭剥削、消除两极分化，最终达到共同富裕。
- 建设社会主义必须从社会主义初级阶段实际出发，坚持改革开放和发展生产力。



**为什么建立和完善社会保障体系是社会主义市场经济体制的重要内容？ (1次)**

- 社会保障体系是市场经济运行的安全网，有利于化解市场竞争带来的风险。
- 它维护社会公平，保障人民基本生活，体现社会主义制度优越性。
- 它有利于稳定社会预期，扩大消费，促进劳动力合理流动。
- 建立多层次社会保障体系，是完善社会主义市场经济体制的重要条件。



**如何理解改革开放是新时期最鲜明的特征？ (1次)**

- 改革是社会主义制度的自我完善和发展，是解放和发展生产力的直接动力。
- 开放是国家繁荣发展的必由之路，有利于吸收世界文明成果、参与国际竞争合作。
- 改革和开放相互促进，共同推动中国实现从封闭半封闭到全方位开放、从计划经济到社会主义市场经济的转变。
- 改革开放不是改旗易帜，而是坚持和发展中国特色社会主义。



**社会主义初级阶段基本经济制度在所有制理论上有哪些突破？ (1次)**

- 确立公有制为主体、多种所有制经济共同发展的基本经济制度。
- 突破了把社会主义等同于单一公有制的传统观念。
- 明确非公有制经济是社会主义市场经济的重要组成部分。
- 强调公有制实现形式可以多样化，一切有利于生产力发展的所有制形式都可以服务于社会主义。



**党在社会主义初级阶段的基本路线是什么？ (1次)**

- 领导和团结全国各族人民，以经济建设为中心。
- 坚持四项基本原则，坚持改革开放。
- 自力更生，艰苦创业。
- 为把我国建设成为富强民主文明和谐美丽的社会主义现代化强国而奋斗。
- 简明概括为“一个中心、两个基本点”。



**什么是社会主义市场经济体制？ (1次)**

- 社会主义市场经济体制，就是使市场在社会主义国家宏观调控下对资源配置发挥基础性作用。
- 计划和市场都是经济手段，不是区分社会主义和资本主义的标志。
- 它把社会主义基本制度同市场经济结合起来。
- 既发挥市场机制效率，又坚持公有制主体地位和共同富裕方向。



**社会主义市场经济体制中公有制为主体如何体现？**

- 公有资产在社会总资产中占优势，这是量的体现。
- 国有经济控制国民经济命脉，对经济发展起主导作用，这是质的体现。
- 公有制主体地位可以有多种实现形式，不要求所有地区、所有行业比例完全一样。
- 坚持公有制主体，是实现共同富裕和社会主义方向的根本保证。



**邓小平理论中社会主义的本质是什么？ (1次)**

- 社会主义的本质，是解放生产力，发展生产力。
- 消灭剥削，消除两极分化。
- 最终达到共同富裕。
- 这一概括把社会主义同发展生产力和人民利益直接联系起来，回答了“什么是社会主义”的核心问题。

- 坚持以经济建设为中心，必须把发展社会生产力摆在首要地位。

- 改革要破除束缚生产力发展的体制机制弊端，推动经济体制、政治体制、文化体制等多方面变革。
- 改革是中国的第二次革命，是解放和发展生产力、实现社会主义现代化的必由之路。
